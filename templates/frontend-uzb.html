Kurs haqida umumiy ma'lumot

Ushbu kurs Javascript asosidagi front-end ilovalarni ishlab chiqishni, xususan, React kutubxonasini o'rganadi. Ushbu kurs React ilovasini ishlab chiqish uchun JavaScript ES6 dan foydalanadi. Siz ham tanishtirasiz Bootstrap 5-ga asoslangan sezgir UI dizayni uchun Reactstrap-dan foydalanish. Siz Reactning turli jihatlari bilan tanishasiz komponentlar. Siz React router va undan bir sahifali ilovalarni ishlab chiqishda foydalanish haqida bilib olasiz. Siz ham o'rganasiz boshqariladigan shakllarni loyihalash haqida. Siz Flux arxitekturasi va Redux bilan tanishasiz. Siz turli xil narsalarni o'rganasiz Redux aspektlari va undan React-Redux quvvatiga ega ilovalarni ishlab chiqish uchun foydalaning. Keyin Fetch uchun foydalanishni o'rganasiz mijoz-server aloqasi va server tomonida REST API-dan foydalanish. React animatsiyani qo'llab-quvvatlash orqali tezkor sayohat va test sinovlari kursni yakunlaydi. Bootstrap bo'yicha mutaxassislik bo'yicha oldingi kursni tugatgan bo'lishingiz kerak 5 yoki ushbu kursni boshqarish uchun Bootstrap 5-da ishlaydigan bilimga ega bo'ling. Bundan tashqari, yaxshi ishlaydigan bilim JavaScript, ayniqsa ES 6 tavsiya etiladi.
